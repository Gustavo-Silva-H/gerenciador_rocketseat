## Gerenciador de Tarefas (Python)

Aplicação de console desenvolvida para auxiliar na organização e controle de atividades diárias, permitindo o registro e acompanhamento de demandas de forma simplificada.

### 1. Sobre o projeto

Este repositório contém o projeto prático desenvolvido durante o primeiro módulo de introdução à linguagem **Python**. O foco principal foi a aplicação de conceitos fundamentais de programação para a construção de uma ferramenta utilitária de linha de comando (CLI) que resolve o problema de organização de tarefas.

O projeto prioriza a legibilidade do código e a estruturação lógica necessária para manipular coleções de dados em tempo de execução.

### 2. Funcionalidades

As funcionalidades implementadas seguem os requisitos de um sistema de gerenciamento básico, incluindo:

*   **Cadastro de Atividades:** Inclusão de novas tarefas na lista de pendências.
*   **Visualização de Tarefas:** Listagem clara de todos os itens registrados.
*   **Atualização de Status:** Possibilidade de marcar tarefas como concluídas ou editar informações existentes.
*   **Remoção de Itens:** Exclusão de atividades da lista conforme a necessidade do usuário.

### 3. Tecnologias utilizadas

O projeto foi construído utilizando as seguintes tecnologias e ferramentas:

*   **Linguagem:** Python 3.x
*   **Ambiente:** Terminal / Console
*   **Versionamento:** Git

### 4. Objetivo do projeto

O objetivo central deste desenvolvimento foi consolidar os conhecimentos adquiridos no início da jornada com Python, especificamente:

1.  Manipulação de variáveis e tipos de dados.
2.  Uso de estruturas de repetição (`for`, `while`) e condicionais (`if`, `else`).
3.  Criação e organização de funções para modularização do código.
4.  Trabalho com estruturas de dados nativas, como listas e dicionários.

### 5. Como executar o projeto

Para rodar a aplicação localmente, siga os passos abaixo:

1.  Certifique-se de ter o **Python 3** instalado em sua máquina.
2.  Clone este repositório:
    `git clone https://github.com/Gustavo-Silva-H/gerenciador_rocketseat.git`
3.  Acesse o diretório do projeto:
    `cd gerenciador_rocketseat`
4.  Execute o script principal (substitua `arquivo_principal.py` pelo nome do arquivo correspondente no repositório):
    `python arquivo_principal.py`
